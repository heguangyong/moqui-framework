# Product Overview

智能推荐 (Smart Recommendation) - An enterprise-grade intelligent recommendation platform built on Moqui Framework with multi-modal AI capabilities.

## Core Features

- **Multi-modal AI**: Speech recognition, image analysis, intelligent dialogue (powered by Zhipu AI GLM-4/GLM-4V)
- **Supply-Demand Matching**: AI-driven intelligent matching and price analysis marketplace
- **Platform Integration**: HiveMind project management, e-commerce, ERP systems

## Main Applications

| Path | Application | Description |
|------|-------------|-------------|
| `/qapps/marketplace` | 智能推荐 | Supply/demand publishing, smart matching |
| `/qapps/tools` | 项目管理 | Task management, collaboration tools |
| `/qapps/system` | 系统管理 | User permissions, system monitoring |
