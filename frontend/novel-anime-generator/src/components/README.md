# Components Directory

This directory contains reusable Vue components for the Novel-to-Anime Video Generator.

## Structure

- `core/` - Core UI components (buttons, inputs, layouts)
- `novel/` - Novel parsing and management components
- `character/` - Character system components
- `episode/` - Episode generation and management components
- `storyboard/` - Storyboard creation and editing components
- `video/` - Video generation and playback components
- `workflow/` - Workflow editor and pipeline components
- `asset/` - Asset library and management components
- `common/` - Common utility components

## Naming Convention

- Use PascalCase for component names
- Prefix with component type when appropriate (e.g., `NovelParser`, `CharacterEditor`)
- Keep names descriptive and specific to their function
