- 📝 小说编辑功能
  - 专用的小说语言支持
  - 语法高亮（章节、角色、对话）
  - 智能代码补全
  - 章节折叠和大纲视图
- 🤖 AI驱动的内容分析
  - 自动章节结构识别
  - 智能角色提取和分析
  - 场景描述解析
- 🎬 工作流编辑器
  - 可视化节点编辑器
  - 拖拽式工作流创建
  - 实时配置验证
  - 工作流模板系统
- 🔧 VS Code 深度集成
  - 活动栏图标和侧边面板
  - 命令面板集成
  - 状态栏显示
  - 上下文菜单
  - 文件关联和图标
- ⚙️ 配置管理
  - AI服务配置
  - 输出格式设置
  - 工作区偏好设置
  - 快捷键自定义
- 🎨 用户界面
  - 现代化的Webview界面
  - 响应式设计
  - 中文本地化
  - 主题适配（明暗模式）

### 技术特性
- TypeScript 严格模式
- 完整的类型定义
- 模块化架构设计
- 错误处理和恢复机制
- 性能优化（懒加载、缓存）
- 自动化构建和发布流程

### 支持的功能
- 小说文件格式：`.txt`, `.md`, `.novel`
- 视频输出格式：MP4, AVI, MOV
- 视频分辨率：720p, 1080p, 4K
- AI模型：GLM-4, GLM-4V-Plus, GLM-3-Turbo

### 已知限制
- 仅支持中文小说内容
- 需要有效的AI服务API密钥
- 大型项目可能需要较长处理时间
- 某些高级功能需要网络连接

### 系统要求
- VS Code 1.74.0 或更高版本
- Node.js 16.x 或更高版本
- 至少 4GB 可用内存
- 网络连接（用于AI服务）

## 版本说明

### 版本号规则
- **主版本号**: 不兼容的API更改
- **次版本号**: 向后兼容的功能新增
- **修订号**: 向后兼容的问题修正

### 更新类型
- **新增**: 新功能
- **更改**: 现有功能的更改
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: 错误修复
- **安全**: 安全相关的修复

### 获取更新
扩展会通过VS Code扩展市场自动更新。你也可以：
1. 打开扩展面板 (`Ctrl+Shift+X`)
2. 搜索 "小说动漫生成器"
3. 点击更新按钮（如果可用）
