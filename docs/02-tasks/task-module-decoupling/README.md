# 🔄 四模块解耦重构任务包

## 📋 任务概述

**目标**: 将四大功能模块完全解耦，移除项目管理与供需匹配的耦合关系
**当前状态**: 📋 **规划完成，待执行**
**预计时间**: 8-12天
**前置条件**: 电商模块完成后启动

## 🎯 核心任务

### **解耦重点**
1. **项目管理独立化** - 移除与supply/demand listing的关联
2. **供需匹配纯化** - 专注B2B商务撮合，不涉及项目创建
3. **模块边界清晰** - 每个模块维护独立的数据模型和服务
4. **协作接口标准化** - 通过API实现松耦合协作

### **技术重构点**
- 移除 HiveMindProject.listingId 字段
- 创建独立的 ProjectInfo 实体
- 重构匹配算法（移除项目创建逻辑）
- 调整 Telegram Bot 菜单结构
- 更新 MCP 路由分类
